第三步 添加依赖库

发送反馈

选择新建的工程“GettingStarted”，右键点击，选择“Properties“，在弹出的对话框中，点击”Java Build Path“>”Libraries“>”Add External JARs…“，添加依赖 的 JAR包（位于【SuperMap iClient for Android 安装目录】\lib中），如”iclient-android-maps-****.jar“（JAR包需要放在当前工程的libs文件夹下），点击”OK“。
