开发环境要求

发送反馈
  1. JDK

版本要求1.6及以上，本示例中使用的是JDK 1.6。

Eclipse 运行需要 Java 运行时环境JRE，包含在JDK（Java Development Kit，Java 开发包）中，而Android SDK需要依赖于JDK，因此需要先准备JDK。

若使用的是非安装版 JDK，需要在系统环境变量中新建一个变量 JAVA_HOME，设置其值为 JDK 的目录，并在系统环境变量Path中添加值“%JAVA_HOME%\bin”和“%JAVA_HOME%\lib”。

  1. Eclipse

版本要求3.6.2及以上（Android SDK要求），本示例中使用的是eclipse 3.6.2版本。

Eclipse 是常用的 Java IDE（Integrated Development Environment，集成开发环境），我们在范例中将使用 Eclipse 作为 Java IDE。

  1. Android SDK

版本要求2.0及以上，本示例中使用的是2.2版本。

Android SDK提供了一系列API库和开发工具，用于在Android平台上构建、测试和调试应用程序。

如果使用的是非安装版Android SDK，解压后需要，环境变量新建一个变量%ANDROID_SDK_HOME%，其值为Android SDK根目录。运行根目录下的“SDK Manager.exe”，可以下载Android平台所需的Platforms和Packages。

  1. ADT插件

版本要求10.0及以上，本示例中使用的是10.0版本。

ADT（Adroid Development Tools for Eclipse）是基于Eclipse IDE进行for Android开发的必须插件，提供用于构建Android应用程序的集成开发环境。

在打开Eclipse后，点击“Help”>“Eclipse Marketplace…”，搜索并安装Android Development Tools for Eclipse。
