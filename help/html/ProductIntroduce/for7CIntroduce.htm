SuperMap iClient 7C 介绍

发送反馈

SuperMap iClient 7C 产品是数据和服务的可视化交互平台，是 SuperMap 服务器系列产品的统一客户端。该产品基于标准的 Web 协议，采用简捷、 易用的面向对象编程模型，提供了功能强大，性能优越，展示效果丰富、扩展性良好的 Web 和 Native 两种开发模式的 SDK 开发包，支持多终端、 跨浏览器，用于构建表现丰富、响应迅速、体验流畅的地图应用。下图展示了 SuperMap iClient 7C 产品体系图：

图1 SuperMap iClient 7C产品体系图

  SuperMap iClient 7C 产品特色

  • 全方位的客户端产品线

    提供 Web、Native 两种开发模式；

    支持 JavaScript、ActionScript、Objective-C、Java、.NET 等主流开发语言在 PC 机、平板电脑、手机等多种终端上的应用开发， 且支持HTML5特性的地图开发，为用户提供丰富选择；

    全面覆盖 iOS、Android、Windows Phone 等不同手机操作系统平台，满足用户从在线应用到移动终端的多样化地图应用。

  • 卓越的可视化效果

    作为地图可视化平台，SuperMap iClient 7C 具备海量数据的快速加载能力和多种类型的数据展现方式，提供的高性能矢量渲染技术， 帮助用户在客户端实现大量动态数据的平滑呈现和良好交互；同时还为用户提供了麻点图、网格图、聚散图、热点图等多种风格的地图可视化手段。

  • 全新的 iServer 服务与其他 WebGIS 共用的连接器

    SuperMap iClient 7C 具备的第三方地图适配功能，打通了各种 WebGIS 与 SuperMap iServer 共用的界限，方便快捷的将 SuperMap iServer 提供的服务与第三方地图数据友好兼容，为用户提供功能强大、体验全新的地图应用。

  • 超强的地图服务聚合能力

    支持 SuperMap iServer 服务及多种标准第三方服务，包括 REST 空间信息服务， WMS、WFS/WFS-T、KML 等 OGC 服务以及 SuperMap 云服务、天地图、Google Maps、百度地图等开放的第三方服务，并支持各种服务在客户端的无缝聚合。

  • 体验优越的二三维一体化

    SuperMap iClient 7C 不仅提供了常规 Web 三维 GIS 系统——SuperMap iClient 7C for Realspace，可与二维 iClient 产品 SDK 无缝聚合, 打造二维三维一体化的客户端系统，而且提供了轻量级 3D GIS 开发平台—— SuperMap iClient 7C for Flash3D，实现了二三维场景的流畅切换，为小场景三维项目提供了良好的解决方案。

  • 支持地图离线缓存

    支持 Localstorage 本地缓存和 MBTiles 离线地图，在网络连接中断的情况下，可浏览之前在线访问过的地图，节省流量； 亦可满足用户在离线状态下的地图应用。

  • 支持多点触控

    支持多点触控，用户可在终端触屏上感受灵活多样的交互体验。

  • 兼容性强，学习成本低

    iClient 7C 系列产品提供了统一的接口，用户在对任意一款产品接口熟悉的基础上，无需重新学习即可短时间内轻松上手，节约学习成本 。

  SuperMap iClient 7C 系列产品：

     SuperMap iClient 7C 提供多个针对不同开发平台的客户端软件开发工具包，包括五套二维开发包和二套三维开发包，打造二维三维一体化的客户端系统。

  • SuperMap iClient 7C for JavaScript

    SuperMap iClient 7C for JavaScript 是一套由 JavaScript 语言编写的 GIS 客户端应用开发包，支持多源数据地图，支持多终端，跨浏览器，通过本产品可快速实现浏览器上美观、流畅的地图呈现

  • SuperMap iClient 7C for Flex

    SuperMap iClient 7C for Flex 是基于 Adobe Flex 技术和 Adobe Flash Builder4 开发平台的 Web 地理信息系统开发包。

  • SuperMap iClient 7C for Silverlight

    SuperMap iClient 7C for Silverlight 是基于 Silverlight Web 开发平台，利用 Microsoft Silverlight 技术和 Microsoft .NET 开发框架实现的富客户端 Web 地图应用开发包。

  • SuperMap iClient 7C for Windows8

    SuperMap iClient7C for Windows 8 包含 Windows Store apps 和 Windows Phone 8 两套开发包，其中 Windows Store apps SDK 是一套基于 Windows Runtime 技术的地理信息系统开发包。利用该套开发包并结合 Windows Runtime 开发框架，可快速方便地构建 Windows 商店风格的地图应用。Windows Phone 8 SDK 是一套基于 Windows Phone 8 的软件开发包，提供了操作简单、功能丰富的地图应用和接口。

  • SuperMap iClient 7C for Android

    SuperMap iClient 7C for Android 是基于 Android 平台开发的轻量级的软件开发包，实现在 Android 平台下快速访问 SuperMap iServer REST地图服务以及 SuperMap Cloud 地图服务。

  • SuperMap iClient 7C for Realspace

    SuperMap iClient 7C for Realspace 是基于 SuperMap UGC（Universal GIS Core）底层类库和 OpenGL 三维图形处理库的三维功能开发包。

  • SuperMap iClient 7C for Flash 3D

    SuperMap iClient 7C for Flash3D 是一套基于 Adobe AIR 技术的web端地理信息系统开发包，不仅可以显示传统的二维地图，还可以展示丰富的三维场景，是一套名副其实的二三维一体化软件开发平台。

  • SuperMap iClient 7C for iOS

    SuperMap iClient 7C for iOS 是一套基于 iOS 平台的轻量级地图软件开发包（SDK），提供了针对 iPhone 以及 iPad 移动设备的 Web 地图访问接口, 包括地图浏览等基本接口，以及查询、量算、标绘等服务，同时支持离线数据的读取，在无网络条件下仍可便捷的访问地图。
