发送反馈

SuperMap iClient 7C for Android 是一套基于Android平台专为移动互联网打造的轻量级软件开发包（SDK），提供了基于Android设备的Web地图应用接口。

  • SuperMap iClient 7C(2015) SP1 for Android
  • SuperMap iClient 7C(2015) for Android
  • SuperMap iClient 7C for Android SP2
  • SuperMap iClient 7C for Android SP1

SuperMap iClient 7C(2015) SP1 for Android

接口变更

  • 规范包名的命名，修改com.supermap.android.ServerType包名为com.supermap.android.serverType；

新增功能

  • 新增支持WMTS可视化的图层WMTSLayerView；

增强内容

  • 优化地图可视化效果，支持各个移动端高低分辨率屏幕的自适应

SuperMap iClient 7C(2015) for Android

接口变更

  • MapView类新增setCacheSize方法设置缓存大小，新增setViewBounds方法设置可视范围；
  • Util类中增加pointInPolygon方法判断一个点是否包含在面内；

新增功能

  • 新增支持WMS可视化的图层WMSLayerView；
  • 新增支持多点平移。

SuperMap iClient 7C for Android SP2

接口变更

  • MapView类新增showScaleControl方法设置比例尺控件是否可见，新增setScaleControlPosition方法设置比例尺控件的布局位置；
  • 软件包com.supermap.android.maps.query新增地图查询服务功能的相关类，ChartQueryService海图查询服务类、QueryByBoundsService范围查询服务类、QueryByDistanceService距离查询服务类、QueryByGeometryService几何查询服务类、QueryBySQLService SQL查询服务类等，详见API文档；

新增功能

  • 新增地图查询服务功能，包括范围、距离、几何和SQL查询服务类，还支持了海图属性查询和海图范围查询；
  • 新增地图比例尺控件，支持设置是否可见和布局位置。

SuperMap iClient 7C for Android SP1

接口变更

  • AbstractTileLayerView类新增setLayerCacheFileName方法；
  • 软件包com.supermap.android.maps新增量算功能的相关类，MeasureMode枚举类、MeasureParameters类、MeasureResult类、MeasureService类、MeasureService.MeasureEventListener类；

新增功能

  • 新增地图量算功能；

修复的问题

  • 解决地图启动加载变慢的问题；
  • 解决浏览地图时出现的闪白问题；
  • 修复不能根据图层名清除该图层缓存的问题；
  • 解决DefaultItemizedOverlay类中清除所有OverlayItem对象，没有清除掉的问题；
  • 删除地图中自带iServer logo的图片；
  • 解决在网络时断时续的情况下，某些瓦片一直出白图的问题；
  • 解决异步初始化图层时，出图或是预处理下载图片会出现崩溃的问题；
  • 修改绘面的默认风格为FILL_AND_STROKE，保证两个点能够绘线，两个点以上绘制填充面；
